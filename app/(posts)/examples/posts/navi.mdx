# NAVI - Tour Guide Agent

NAVI is an innovative tour guide agent that combines TypeScript and Python with AI technologies to create an intelligent travel companion. This project showcases the intersection of modern web technologies with artificial intelligence to deliver personalized tour experiences.

## Features

- AI-powered tour recommendations
- Interactive user interface
- Real-time navigation assistance
- Personalized travel insights

## Technologies Used

- TypeScript
- Python
- AI/ML frameworks
- Modern web technologies
